# 🩸 BloodBridge – Real-Time Blood Donation Platform

🚨 Finding blood during emergencies can be stressful and time-sensitive.  
**BloodBridge** is a real-time blood donation platform built using the MERN stack to connect blood donors and recipients instantly.

---

## 🌐 Live Demo
👉 https://blood-donate-front.vercel.app

---

## ✨ Features

- 🧑‍🤝‍🧑 User registration with blood group details
- ⚡ Real-time blood request notifications using Socket.IO
- 👀 Detailed request view page
- 🤝 Option to respond: Donate or refer someone

---

## 🖥️ Tech Stack

- MongoDB
- Express.js
- React.js
- Node.js
- Socket.IO
- JWT Authentication
- Vercel (Frontend Deployment)
- Render (Backend Deployment)

## ⚙️ How It Works

1. Users sign up and provide their blood group
2. A user creates a blood request (e.g., O+)
3. Users receive instant notifications via Socket.IO
4. Users click notification → redirected to Requests page
5. They can open request details and respond
6. Once fulfilled, request is resolved and removed by the Requested Person
